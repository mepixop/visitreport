## Description

Repository for visit report project for IHK.

## Project setup

```bash
$ npm install
```

You must also create a .env file with the following values:

```
DATABASE_USER=
DATABASE_PASSWORD=
DATABASE_NAME=
DATABASE_HOST=
```

An empty database with the name 'visitReport' can be created by running the accompanying setup.sql file.

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Run tests

```bash
# unit tests
$ npm run test
```
